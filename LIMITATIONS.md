# このテンプレートの限界と課題

## 主な課題

### 1. 初期コストが高い
- 50ファイルのテンプレート
- どこから手をつけるか不明
- ドキュメント作成が目的化するリスク

### 2. 自動参照されない
- ユーザーが毎回「このドキュメントを読んで」と指示する必要がある
- `CLAUDE.md` との連携が未設計

### 3. ドキュメントとコードの同期
- コードが変わればドキュメントは陳腐化
- 更新する動機付けがない
- 乖離は必ず起きる

### 4. 実例がない
- プレースホルダーだらけ
- 「こう書くとAIが理解しやすい」という具体例がない

## 改善の方向性

| 課題 | 対策案 |
|------|--------|
| 量が多い | 必須3-5ファイルの「ミニマム版」 |
| 自動参照なし | CLAUDE.md に参照ルールを明記 |
| 同期問題 | コードからドキュメント生成（OpenAPI等） |
| 実例なし | サンプルプロジェクトで埋めた例を提供 |
