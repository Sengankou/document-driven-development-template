# デザインシステム

このドキュメントでは、UI設計の基本方針とデザインガイドラインを定義します。

## 基本方針

### デザイン原則
1. **一貫性**: 同じ要素は同じ見た目と振る舞い
2. **シンプルさ**: 必要最小限の要素でタスクを達成
3. **アクセシビリティ**: すべてのユーザーが利用可能

### 使用するデザインシステム/ライブラリ
- （例: Material Design, Tailwind CSS, 独自デザインシステム など）

## レスポンシブデザイン

### ブレークポイント
| 名称 | 幅 | 対象デバイス |
|------|-----|-------------|
| xs | < 640px | モバイル |
| sm | >= 640px | 大きめのモバイル |
| md | >= 768px | タブレット |
| lg | >= 1024px | 小さめのデスクトップ |
| xl | >= 1280px | デスクトップ |

### モバイルファースト
- 基本スタイルはモバイル向け
- より大きな画面向けにスタイルを追加

## アクセシビリティ

### 準拠レベル
- WCAG 2.1 Level AA

### 要件
- [ ] カラーコントラスト比 4.5:1 以上（通常テキスト）
- [ ] カラーコントラスト比 3:1 以上（大きなテキスト）
- [ ] キーボードナビゲーション対応
- [ ] スクリーンリーダー対応
- [ ] フォーカスインジケーター表示
