## 優先度の定義

| 優先度 | 定義 |
|--------|------|
| 高 | ビジネス上必須、早期に実装すべき |
| 中 | 重要だが、後回しにできる |
| 低 | あると良いが、なくても問題ない |

## ステータスの定義

| ステータス | 定義 |
|-----------|------|
| 検討中 | まだ詳細が決まっていない |
| 採用 | 実装が決定、フェーズに組み込み |
| 却下 | 実装しないことが決定 |
| 完了 | 実装済み |
