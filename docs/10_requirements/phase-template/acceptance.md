## 非機能要件の受け入れ基準

### パフォーマンス
| 項目 | 基準 | 計測方法 |
|------|------|---------|
| ページ読み込み | 3秒以内 | Lighthouse |
| API レスポンス | 200ms 以内（95パーセンタイル） | APM ツール |

### セキュリティ
| 項目 | 基準 | 検証方法 |
|------|------|---------|
| 認証 | 正しい認証情報以外でアクセス不可 | セキュリティテスト |
| データ保護 | 機密データが暗号化されている | コードレビュー |

### アクセシビリティ
| 項目 | 基準 | 検証方法 |

| WCAG | Level AA 準拠 | axe / Lighthouse |
