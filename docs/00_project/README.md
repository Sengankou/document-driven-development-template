# 00_project - プロジェクト定義

プロジェクト全体に関わる基本情報を定義するドキュメント群です。

## 特性
- **累積的**: フェーズを超えて継続的に参照・更新される
- **対象読者**: 全ステークホルダー

## ドキュメント一覧

| ファイル | 目的 | 更新頻度 |
|---------|------|---------|
| [charter.md](charter.md) | プロジェクトの目的・スコープ・成功基準 | 初期に作成、大きな方針変更時に更新 |
| [organization.md](organization.md) | 組織体制・役割・責任 | 体制変更時に更新 |
| [glossary.md](glossary.md) | プロジェクト固有の用語定義 | 継続的に追加 |
| [risks.md](risks.md) | リスク管理表 | 継続的に更新 |

## 作成順序

1. **charter.md** - 最初に作成（プロジェクト開始時）
2. **organization.md** - charter と並行して作成
3. **glossary.md** - 初版を作成し、以降継続的に追加
4. **risks.md** - 初版を作成し、以降継続的に更新
